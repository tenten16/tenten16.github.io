This site is basically a blank canvas for quickly building a Jekyll site. This is a pretty bias setup. Feel free to do with it whatever you like.
I would personally start by deleting the helloworld.html file and removing the include from the index.html

This Jekyll setup is powered by Gulp for Sass, autoprefixer, and browsersync
